Web Browser

Definition:

This is a software application that used to acces, view and display websites

Examples of Web Browsers:

  • Google Chrome
  • Mozilla Firefox
  • Safari
  • Microsoft Edge

Key Functions

  • Requests web pages from servers
  • Displays HTML, CSS, and JavaScript
  • Allows users to navigate using links
  • Stores history and bookmarks

How It Works (Simple)

When you type a website address, the browser sends a request over the internet. The server responds with the webpage, and the browser displays it for you.

Example

When you open Chrome and type "www.google.com", the browser loads the Google homepage.
