Hypertext Transfer Protocol (HTTP)

Definition

HTTP (HyperText Transfer Protocol) is a set of rules used by web browsers and web servers to communicate and exchange information on the World Wide Web.

How HTTP Works

  • The browser sends an HTTP request to a web server
  • The server processes the request
  • The server sends back an HTTP response
  • The browser displays the web page to the user

Key Points

  • HTTP stands for HyperText Transfer Protocol
  • It defines how data is requested and sent over the web
  • It works between web browsers and web servers
  • HTTPS is the secure version of HTTP

Example

When you open a website like Google, your browser uses HTTP (or HTTPS) to request the page and receive the content from the server.
