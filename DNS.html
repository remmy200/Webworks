Domain Name System (DNS)

Definition:

The Domain Name System translates human readable names such as www.example.com into numerical IP addresses that computers can use to identify each other on the network.

How DNS Works (Simple Steps)

  • You type a website name (URL) in your browser
  • The browser asks DNS for the IP address
  • DNS looks up the correct IP address
  • DNS sends the IP address back to the browser
  • The browser connects to the web server using that IP

Key Points

  • DNS stands for Domain Name System
  • It converts domain names into IP addresses
  • It makes the internet easier for humans to use
  • Without DNS, we would need to memorize numbers instead of names

Example

When you type "www.youtube.com", DNS converts it into an IP address so your browser can find YouTube’s server.
