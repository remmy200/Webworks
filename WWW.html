World Wide Web

Definition:

The World Wide Web commonly referred to as the Web(www)is an informational system where documents, videos, images and other web resources are identified by Uniform Resource Locators (URLs), interlinked by hypertext links and can be via the internet.

The Internet was invented by Tim Berners-Lee in 1989, and has become a fundamental part of modern communication and information sharing.

Key Points

  • It uses web browsers like Chrome or Firefox
  • Web pages are written using HTML
  • It uses HTTP/HTTPS to transfer data
  • It connects billions of web pages worldwide

Example:

When you visit a website like google or youtube, you are using the World Wide Web.

A format in which people use to look for webpages is by typing www followed by the website name.
