Multitier Architecture

Definition

Multitier architecture is a system design approach where an application is divided into separate layers (tiers), each with a specific role.

Main Tiers

  • Presentation Tier: The user interface (what you see in the browser)
  • Application Tier: Processes data and handles logic
  • Data Tier: Stores and manages data (databases)

How It Works

1. The user interacts with the website (Presentation Tier)
2. The request is sent to the server logic (Application Tier)
3. Data is retrieved from the database (Data Tier)
4. The response is sent back to the user

Key Points

  • Separates system into layers for better organization
  • Makes systems easier to manage and update
  • Improves security and performance
  • Common in modern web applications

Example

When you log into a website: your login page (front-end) sends data to the server (logic), which checks the database and returns your account information.
