Web Server

Definition:

A web server is a computer system that delivers web pages to users upon requests. Computers can store, process and deliver web pages to users, when a Uniform Resource Locator (URL) is sent to to a browser, the browser then sends a request to the web server which then sends back the web pages to the user requesting them.

Key Functions

  • Stores website files (HTML, CSS, images)
  • Processes requests from web browsers
  • Sends web pages back to users
  • Handles multiple users at the same time

How It Works

1. You enter a website address in your browser
2. The browser sends a request to the web server
3. The server processes the request
4. The server sends the webpage back to your browser
5. The browser displays the page

Example

When you visit a website like Google, your browser sends a request to Google's web server, which then sends back the webpage you see.
