Extranet

Definition

An extranet is a controlled private network that allows external users (such as customers, suppliers, or partners) limited access to an organization’s internal systems.

Key Features

  • Partially private network
  • Used by organizations to share data with outsiders
  • More secure than the internet but less restricted than an intranet
  • Requires authentication (login access)

Extranet vs Intranet vs Internet

  • Internet: Public access for everyone
  • Intranet: Private, only internal users
  • Extranet: Private but shared with selected external users

Example

A company may use an extranet so suppliers can track orders and deliveries, while customers can check order status online.
