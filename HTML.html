HyperText Markup Language

Definition

HTML (HyperText Markup Language) is the standard language used to create and structure web pages on the internet.

Simple Explanation

Think of HTML like the skeleton of a human body. It gives structure to a webpage, such as headings, paragraphs, images, and links, but does not control how it looks.

What HTML Does

  • Creates the structure of web pages
  • Defines headings, paragraphs, and links
  • Displays images and multimedia content
  • Works with CSS and JavaScript to build full websites

Basic Example

A simple HTML element looks like this: 

<h1>Hello World</h1>

This creates a main heading on a web page.

Key Points

  • HTML stands for HyperText Markup Language
  • It is not a programming language (it is a markup language)
  • It is the foundation of all websites
  • Browsers read HTML to display web pages

Example in Real Life

Every website you visit (Google, YouTube, Facebook) is built using HTML to structure the content you see.
