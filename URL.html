Uniform Resource Locator (URL)

Definition:

A Uniform Resource Locator is a unique address used to locate and access specific resources on the internet.

Structure of a URL

Example: https://www.example.com/page

  • https:// – The protocol (how data is transferred)
  • www.example.com – The domain name (website name)
  • /page – The specific page or resource

Key Points

  • It identifies the location of a resource on the internet
  • It is used by web browsers to access websites
  • Works together with DNS to find the correct server
  • Usually starts with HTTP or HTTPS

Example

When you type "https://www.google.com" into your browser, you are using a URL to access Google's website.
