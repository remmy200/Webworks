Web 2.0

Definition

Web 2.0 refers to the second generation of the World Wide Web, where users can interact, create, and share content online.

Key Characteristics

  • Interactive and user-generated content
  • Social media platforms (Facebook, YouTube, Instagram)
  • Users can comment, like, and share
  • Dynamic and constantly updated content

How It Works

Users interact with websites by uploading content or responding to others. The website processes this input and updates content in real time or near real time.

Examples

  • YouTube (users upload videos)
  • Facebook (posts, comments, likes)
  • Wikipedia (users edit content)

Key Point

Web 2.0 is known as the “interactive web”.
